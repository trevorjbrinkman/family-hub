Setup
You need two things: a Google API Key (free) and your Calendar ID. Your calendar must be set to Public in Google Calendar settings. See the README for step-by-step instructions.

To add prep items to an event, include a line in the event description starting with pack: or prep: — e.g.
pack: shin pads, water bottle, snacks

Usually your Gmail address. Find it in Google Calendar → Settings → your calendar → Calendar ID.
Free key from console.cloud.google.com — enable the Google Calendar API.
